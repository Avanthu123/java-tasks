# Java Tasks

This repository contains Java tasks demonstrating various concepts.

## Tasks

### Task 1: Bank Account
- **Description**: Demonstrates OOP principles (Encapsulation, Inheritance, Polymorphism, Interface).
- **Files**: `t1/BankAccount.java`
- **Output**:
```
2000.0 deposited.
1000.0 withdrawn.
Balance: 6000.0
----------------------
Balance with interest: 10500.0
```
---

### Task 2: Student Grade Calculator
- **Description**: Calculates student grades based on marks using operators and conditions.
- **Files**: `t2/StudentGradeCalculator.java`
- **Output**:
```
Enter student name: Brittany
Enter marks for subject 1: 35
Enter marks for subject 2: 89
Enter marks for subject 3: 90

Student Name: Brittany
Total Marks: 214.0
Average Marks: 71.33333333333333
Grade: C
```
---

### Task 3: Static & Access Modifiers
- **Description**: Demonstrates static variables/methods, instance tracking, and access modifiers (`private`, `protected`, `public`).
- **Files**: `t3/Counter.java`, `t3/CounterDemo.java`
- **Output**:
```
Counter: Alpha | ID: 101 | Value: 50.5
Counter: Beta | ID: 102 | Value: 75.0
Counter: Gamma | ID: 103 | Value: 100.2
Total instances created: 3
```
---

### Task 4: Array & String Logic 
- **Description**: Implementation of array reversal, second largest element, and palindrome checking.
- **Files**: 
  - `t4/ArrayReverser.java`
  - `t4/SecondLargestFinder.java`
  - `t4/PalindromeChecker.java`
- **Output**:
```
Original Array: [10, 22, 31, 70, 50]
Reversed Array: [50, 70, 31, 22, 10]
Second Largest: 50
Is 'Racecar' palindrome? true
```

---

### Task 5: Collections & Generics 
- **Description**: Separated generic class logic from collection usage.
- **Files**:

  - `t5/Box.java`
